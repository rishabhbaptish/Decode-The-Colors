Decode the Colors

How to Play

1. Select a color from the palette. Drag and drop the selected color into one of the circles.
2. After filling all the circles, click the Submit button.
3. Feedback will be displayed in the Move Order pane. It shows which colors and positions are correct.
4. Use this as a clue for your next guess.
5. Each level has a set number of moves. If you exceed the allowed moves without solving the puzzle, you will have to retry the level.
6. When you correctly guess the color sequence, you can then proceed to the next level.
